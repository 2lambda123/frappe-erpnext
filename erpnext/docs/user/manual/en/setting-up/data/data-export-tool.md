# Data Export Tool

Use this tool to export data in Excel or CSV format

To open the Data Export Tool, you can either go to Setup > Data Export OR use global search


<img alt="Data Export" class="screenshot" src="/Users/sps/foobar/apps/erpnext/erpnext/docs/assets/img/setup/data-export/data-export-tool.gif">

### Options

1. Select Doctype

    You will be provided with the exportable document type options.

2. File Type

    Select File format of the data to be exported

3. Add Filter (optional)

    Add filters to export only required data.

    eg. If you want the data of project related to specific company, you can use the filter to select the company.

4. Field Multicheck (optional)

    Only checked fields will be exported.

**Note:**

    If you do not want a child table to be exported simply deselect all the fields of that child table.
