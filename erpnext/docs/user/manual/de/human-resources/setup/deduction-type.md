## 11.12.7 Abzugsart

Sie können zu Steuern oder anderen Abzügen vom Gehalt Datensätze erstellen und diese als Abzug beschreiben.

Um eine neue Abzugsart anzulegen, gehen Sie zu:

> Personalwesen > Einstellungen > Abzugsart > Neu
